== Description ==
This WordPress plugin is Easy Digital Downloads add-on/extension, adds 'Website' and 'Extra note(s)' custom fields to the checkout screen.

= This plugin will... =
* Adds 'Website' and 'Extra note(s)' custom fields to the checkout screen
* Shows the 'Website' field on the "view order details" screen, and
* Adds a new {website} and {extra_note} email tag to show the 'Website' and 'Extra note' in either the standard purchase receipt or admin notificatios

- This plugin doesn't have any settings. Install - Activate, and Boom!

**Follow the development of this plugin on [GitHub](https://github.com/dipakcg/edd-custom-fields-on-checkout)**

**P.S. It is always the best policy to open a [support thread](http://wordpress.org/support/plugin/edd-custom-fields-on-checkout) first before posting any negative review.**

== Installation ==
1. Upload the ‘edd-custom-fields-on-checkout’ folder to the ‘/wp-content/plugins/‘ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. That’s it!
